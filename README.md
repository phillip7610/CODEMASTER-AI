# CodeMaster AI 🚀

Plateforme d'apprentissage progressif et interactive pour apprendre la programmation avec une IA tutorielle personnalisée.

## 🎯 Objectif Principal

Transformer des débutants en développeurs compétents grâce à :
- **Leçons structurées** par niveaux progressifs
- **Exercices interactifs** avec correction automatique
- **IA tutorielle** adaptée au niveau de l'utilisateur
- **Système de gamification** pour maintenir la motivation
- **Mémoire intelligente** des erreurs et forces
- **Projets réels** pour appliquer les connaissances

## 📚 Langages Disponibles

- HTML, CSS, JavaScript
- Python, Java, C, C++, C#, PHP, SQL

## 🎓 Niveaux d'Apprentissage

Chaque langage possède 5 niveaux :

1. **Simple** - Bases absolues (variables, affichage, premières fonctions)
2. **Facile** - Conditions, boucles simples, fonctions
3. **Moyen** - Tableaux, objets, algorithmes
4. **Difficile** - Structures complexes, optimisation, POO, API
5. **Impossible** - Projets réels, algorithmes avancés, défis techniques

## 🛠️ Technologies

### Frontend
- React 18+, Tailwind CSS, CodeMirror

### Backend
- Node.js + Express, MongoDB, JWT, OpenAI API

## ⚡ Démarrage Rapide

```bash
git clone https://github.com/phillip7610/CodeMaster-AI.git
cd CodeMaster-AI
npm run install-all
cp .env.example .env
npm run dev
```

Frontend: `http://localhost:3000`
Backend: `http://localhost:5000`
